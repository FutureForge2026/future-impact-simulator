# Global Future Impact Simulator

Global Future Impact Simulator is a web application that combines World Bank economic indicators and ESCO occupational classification data to help users explore career opportunities across different countries.

The simulator allows users to:

* Select a country
* Enter an occupation
* Provide a salary estimate
* View related occupations using ESCO classifications
* Explore economic indicators from World Bank datasets
* Generate a country opportunity score

## Why I Built This

Many people compare career opportunities based primarily on salary.

However, salary alone rarely provides the full picture.

This project was built to combine occupational and economic data into a single tool that provides additional context when evaluating opportunities across countries.
